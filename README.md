
This is a Lua (Luau) script designed for Roblox. It features a clean Graphical User Interface (GUI) that allows players to select one or multiple targets from the server and continuously pull their hitboxes (`HumanoidRootPart`) right in front of the local player's character. 

This script is primarily intended for testing local attack hitboxes or replicating a "Hitbox Magnet" / "Kill Aura" effect.
